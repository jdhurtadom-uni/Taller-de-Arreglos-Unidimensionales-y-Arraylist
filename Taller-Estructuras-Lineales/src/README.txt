# Taller de arreglos, arraylist y matrices Juan Diego Hurtado Molano



## Descripción del Taller
Este proyecto es una implementación en Java (ejecutada desde consola) para resolver problemas prácticos utilizando los conocimientos de la clase de estructuras de datos lineales: arreglos unidimensionales, matrices (bidimensionales) y listas dinámicas (ArrayList).El código está diseñado siguiendo los principios de Clean Code, Programación Orientada a Objetos y principios SOLID (DRY, KISS, YAGNI).

## Instrucciones de Ejecución
1. Clonar el repositorio.
2. Abrir la carpeta `Taller-Estructuras-Lineales` en Visual Studio Code.
3. Asegurarse de tener instalado el "Extension Pack for Java".
4. Ejecutar los diferentes puntos del taller que se encuentran cada uno en una carpeta asignada dependiendo del tema, cadauno cuenta con su main dentro de cada clase al entrar a cualquiera de los ejercicios solo se deberá correr el programa y seguir las instrucciones.
